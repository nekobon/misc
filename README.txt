Miscellaneous small programs/code.

- decos.py
  Contains some python decorators.

- html_parser.py
  A primitive html parser.

- downloader.py
  A python function to download files.

- sendmail.py
  A python wrapper to use linux sendmail to send an email.

- vectoring.py
  A 3D vector class with overridden operators.

- spinanimation.m
  A matlab file to create and save an animation of an unit vector
  rotating in a meshed unit sphere.
